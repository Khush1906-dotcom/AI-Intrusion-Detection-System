# AI-Intrusion-Detection-System
This project is an AI-based cybersecurity system that detects whether network traffic is normal or malicious (attack).

It is built using a dataset of network activities, where different features like duration, bytes transferred, and connection patterns are analyzed. The data is first cleaned and converted into numerical form, then a machine learning model (Random Forest) is trained to learn patterns of normal and attack traffic.

Once trained, the system can take new input and predict in real time whether the activity is safe or suspicious. It also includes an interactive interface for user input and displays results clearly as Normal Traffic or Attack Detected.
